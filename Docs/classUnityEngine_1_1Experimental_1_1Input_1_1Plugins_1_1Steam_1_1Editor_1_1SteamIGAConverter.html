Detailed Description

Converts input actions to and from Steam IGA file format.

The idea behind this converter is to enable users to use Unity's action editor to set up actions for their game and the be able, when targeting desktops through Steam, to convert the game's actions to a Steam VDF file that allows using the Steam Controller API with the game.

The generated VDF file is meant to allow editing by hand in order to add localization strings or apply Steam-specific settings that cannot be inferred from Unity input actions.

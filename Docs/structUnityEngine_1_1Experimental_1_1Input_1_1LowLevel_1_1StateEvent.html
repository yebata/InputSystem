Detailed Description

This is a variable-sized event.

Read the current state of device and create a state event from it.

Parameters
deviceDevice to grab the state from. Must be a device that has been added to the system.
eventPtrReceives a pointer to the newly created state event.
allocatorWhich native allocator to allocate memory for the event from. By default, the buffer is allocated as temporary memory (Allocator.Temp. Note that this means the buffer will not be valid past the current frame. Use Allocator.Persistent if the buffer for the state event is meant to persist for longer.
Returns
Buffer of unmanaged memory allocated for the event.
Exceptions
ArgumentExceptiondevice has not been added to the system.
ArgumentNullExceptiondevice is null.

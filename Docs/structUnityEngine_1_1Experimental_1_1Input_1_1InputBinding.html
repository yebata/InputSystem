Detailed Description

A mapping of control input to an action.

A single binding can match arbitrary many controls through its path and then map their input to a single action.

A binding can also be used as a override specification. In that scenario, path, action, and groups become search criteria that can be used to find existing bindings, and overridePath becomes the path to override existing binding paths with.

Enumerator
ThisAndNextCombine 

This and the next binding in the list combine such that both need to be triggered to trigger the associated action.

The order in which the bindings trigger does not matter.

An arbitrarily long sequence of bindings can be arranged as having to trigger together.

If this is set, ThisAndPreviousCombine has to be set on the subsequent binding.

Composite 

Whether this binding starts a composite binding group.

This flag implies PushBindingLevel. The composite is comprised of all bindings at the same grouping level. The name of each binding in the composite is used to determine which role the resolved controls play in the composite.

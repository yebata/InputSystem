◆ PointerDeltaSensitivity

float PointerDeltaSensitivity = 0.25f
static

Default sensitivity for pointer deltas.

Pointer deltas flow into the system in pixel space. This means that the values are dependent on resolution and are relatively large. Whereas a gamepad thumbstick axis will have normalized values between [0..1], a pointer delta will easily be in the range of tens or even hundreds of pixels.

Pointer sensitivity scaling allows to turn these pointer deltas into useful, partially resolution-independent floating-point values.

The value determines how much travel is generated on the delta for each percent of travel across the window space. If, for example, the mouse moves 15 pixels on the X axis and -20 pixels on the Y axis, and if the player window is 640x480 pixels and the sensitivity setting is 0.5, then the generated pointer delta value will be (15/640*6, -20/480*6) = (0.14, -0.25).
